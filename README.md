# BMI Calculator (VB.NET)

**Description:**  
A simple **VB.NET Windows Forms** project to calculate Body Mass Index (BMI).  
Users can input their weight and height to calculate their BMI and get a classification (Underweight, Normal, Overweight, Obese).

---

## **Features**
- Simple and user-friendly interface using Windows Forms  
- Input weight (kg) and height (m) easily  
- Automatic BMI calculation  
- Display BMI result with classification  

---

## **Requirements**
- Visual Studio 2017 or later  
- .NET Framework 4.7.2 or later  

---

## **How to Run**
1. Open the solution file `BMI_Calculator.sln` in Visual Studio  
2. Run the project using **Start**  
3. Enter your weight and height, then click **Calculate** to get your BMI and classification  

---

## **Example Usage**
| Weight (kg) | Height (m) | BMI  | Classification |
|-------------|------------|------|----------------|
| 70          | 1.75       | 22.9 | Normal weight  |
| 90          | 1.75       | 29.4 | Overweight     |
| 50          | 1.75       | 16.3 | Underweight    |
| 110         | 1.75       | 35.9 | Obese          |
